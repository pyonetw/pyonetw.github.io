Python 小抄

項目 說明
輸出與輸入
input('提示訊息')提示訊息必須為一個字串。任何輸入結果均為字串
print('顯示訊息')顯示一個字串訊息
print('顯示訊息1', '顯示訊息2')可用英文逗號 , 分隔兩個以上資料或變數
print('顯示訊息 {}'.format(var1))以 format() 套用資料或變數
print('顯示訊息 {} {}'.format(var1, var2))以 format() 套用兩個以上資料或變數，參數用英文逗號 , 分隔
print('顯示訊息 %s' % (var1))以 % 套用資料或變數
print('顯示訊息 %s %s' % (var1, var2))以 % 套用兩個以上資料或變數，參數用 tuple 設定
算術
int('87')整數字串或轉整數值；取出浮點數的整數部分
float('1.38')浮點數字串轉浮點數值；將整數改用浮點數表示，例如 123 轉為 123.0
a + b兩數相加
a - b兩數相減
a * b兩數相乘
a / b兩數相除
a // b兩數相除取商數
a % b兩數相除取餘數；同餘運算
a ** b指數，或稱次方。2**4 = 2*2*2*2
註解與變數
註解作用說明片段程式或特定變數的意義、用途或流程；註記程式相關問題
#單行註解。在 repl.it, VS Code 或 ST3 中，可選取多行，[Ctrl]+[/] 註解或反註解
''' 或 """多行註解。三單引號或三雙引號，必須前後成對
變數作用存取資料
「存」變數資料使用一個等號，變數名稱設於左邊。例如：a = 1 + 2，為將 1 + 2 的結果存於變數 a
「取」變數資料取變數資料，指定變數名稱即可。例如：b = a + 3，為先取出變數 a 的值，和 3 相加後，存於變數 b
i = 123設變數 i 為整數數值資料
f = 12.34設變數 f 為浮點數數值資料
s = 'abc123'設變數 s 為字串資料
tup = (1, 2, 3)設變數 tup 為 tuple (1, 2, 3) 資料
lst = ['a', 'b', 'c']設變數 lst 為 list ['a', 'b', 'c'] 資料
比較運算
比較運算通常以兩個資料或變數相互比較。比較結果「是」為 True，「否」則為 False
比較結果可理解為「條件成立」為 True (真)，「條件不成立」為 False (假)
==是否等於
!=是否不等於
>是否大於
>=是否大於等於
<是否小於
<=是否小於等於
決策式
if 條件式: 單決策式。條件成立，執行 if 內程式
if 條件式: - else: 雙決策式。條件成立，執行 if 內程式；條件不成立，執行 else 內程式
if 條件式: - elif 條件式: -...- elif: 多決策式。依序判斷，任一 if 條件成立，執行相關程式後，離開 if 區段
if 條件式: - elif 條件式: -...- elif: - else: 多決策式。依序判斷，任一 if 條件成立，執行相關程式後，離開 if 區段；都不成立，執行 else 內程式
if a==1 and b==2: 判斷兩個條件以上，所有條件成立，執行相關程式
if a==1 or b==2: 判斷兩個條件以上，任一條件成立，執行相關程式
if a in (1, 2, 3):判斷變數 a 的內容是否為 tuple 資料其中之一
if a in [1, 2, 3]:判斷變數 a 的內容是否為 list 資料其中之一
if s in '0123456789abcdef':判斷變數 s 的內容是否為字串資料其中之一部分
迴圈式
for i in range(n):迴圈執行 n 次，依序讀取數值 0 ~ n-1
for i in range(1, n+1):迴圈執行 n 次，依序讀取數值 1 ~ n
for i in range(start, stop, step):start 起始值，stop 終止值，step 間隔值
for i in (1, 2, 3):迴圈讀取 tuple 資料，執行次數依 tuple 資料個數而定
for i in [1, 2, 3]:迴圈讀取 list 資料，執行次數依 list 資料個數而定
while 布林變數:while 判斷布林變數，為 True 才進入迴圈
while 條件式:while 判斷條件式，條件成立，結果為 True 才進入迴圈
while True:直接設 True 進入迴圈
while 1: 等於用 True，進入迴圈
while 0: 等於用 False，不進入迴圈
break離開迴圈，迴圈內任何地方均可 break。可用於 for 或 while
continue忽略 continue 之後所有程式，直接跳到迴圈結構句從頭執行。可用於 for 或 while
容器與資料切取
tuple 小括或圓括 (1, 2, 3)。資料不可變更，即不可新增、修改或刪除
list 中括或方括 [1, 2, 3]。新增 append()、修改、刪除 del
dict 大括或花括 {'key1':val1, 'key2':val2}。可新增、修改或刪除
str[index]切取字串中索引值為 index 的字元
str[start:stop:step]切取部分字串
tuple[index]切取 tuple 中索引值為 index 的資料
tuple[start:stop:step]切取 tuple 中部分資料
list[index]切取 list 中索引值為 index 的資料
list[start:stop:step]切取 list 中部分資料
(1, 2, 3, 4, 5).index(3)取得資料 3 在 tuple 中的 index 值
[1, 2, 3, 4, 5].index(3)取得資料 3 在 list 中的 index 值
彙總函式
len('something')計算字串長度，即計算字元個數
len((1, 2, 3, 4, 5))計算 tuple 中資料個數
len([1, 2, 3, 4, 5])計算 list 中資料個數
sum()加總 tuple 或 list 中，所有數值資料
max()找出 tuple 或 list 中，最大數
min()找出 tuple 或 list 中，最小數
亂數函式
random.randrange(1, 11)亂數取 1~10 整數
random.randrange(start, stop, step)亂數範圍依：start 起始值、stop 終止值、step 間隔值而定
random.randint(1, 100)亂數取 1~100 整數
random.choice(['a', 'b', 'c'])亂數從 list 中選取任一個資料。可改用於 tuple
random.shuffle(one_list)list 資料洗牌；不可用於 tuple 或字串
random.sample(取樣來源, 取樣個數)從 list, tuple 或字串中隨機取樣，取樣結果一律存於 list
其它函式
'str'.title()全部單字首字母大寫
'str'.upper()全部字母大寫
'str'.lower()全部字母小寫
'str'.split()單字串切為多字串，可指定次數；結果存於 list
'str'.rsplit()單字串切為多字串，可指定次數 (從右切割)；結果存於 list
'間隔字串'.join(tuple_or_list)多字串併為單字串；須先指定間隔字串，可為空字串
'str'.count()計算小字串在大字串中個數；計算指定資料在容器中的個數
'str'.zfill()依指定字元寬度補 0
'str'.strip()兩邊移除指定字串
'str'.find()尋找指定字串；找到回傳第一個 index 值，找不到回傳 -1
'str'.replace()取代指定字串
'12ab'.isalnum()判斷是否含字母及數字；結果為 True 或 False
'abcd'.isalpha()判斷是否僅含字母；結果為 True 或 False
'1234'.isdigit()判斷是否僅含數字；結果為 True 或 False
'ABCD'.isupper()判斷是否全為英文大寫字母；結果為 True 或 False
'abcd'.islower()判斷是否全為英文小寫字母；結果為 True 或 False
